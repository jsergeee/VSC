Инструкция по установке и запуску:
Создайте директорию проекта (если ещё не создана):

text
C:\Users\User\Documents\VSC\python_project\event_log\
Создайте все 6 файлов в этой директории с указанным выше содержимым.

Установите необходимые модули (если ещё не установлены):

bash
pip install pywin32
Запустите программу от имени администратора:

bash
python main.py
Структура проекта:
main.py - основной исполняемый файл

event_database.py - база данных событий

cve_database.py - база данных CVE уязвимостей

utils.py - вспомогательные функции

css_style.py - CSS стили для отчета

html_generator.py - генератор HTML отчетов
