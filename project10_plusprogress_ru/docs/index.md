# Добро пожаловать в PlusProgress!

**Онлайн школа английского языка** — проект для управления уроками, учениками и учителями.

## 🎯 Возможности проекта

- 👨‍🏫 **Управление учителями** — добавление, редактирование, привязка предметов
- 👨‍🎓 **Управление учениками** — регистрация, личные кабинеты, баланс
- 📚 **Уроки** — создание, расписание, отчеты
- 💰 **Финансы** — платежи, выплаты учителям, депозиты
- 📝 **Домашние задания** — выдача, сдача, проверка
- 📄 **Методические материалы** — загрузка, доступ

## 🔧 Технологии

- **Backend:** Django 5.2, Django REST Framework 3.16
- **База данных:** SQLite (разработка), PostgreSQL (продакшн)
- **Аутентификация:** Token-based
- **Документация:** Swagger UI + MkDocs
